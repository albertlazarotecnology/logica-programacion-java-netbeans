# Mini proyecto 1: Boleta simple de venta

Este mini proyecto fue realizado como práctica del curso:
"Lógica Programación: Aprende Programar en Cualquier Lenguaje".

## Objetivo

Practicar el uso de variables, constantes y operaciones básicas en Java.

## ¿Qué hace el programa?

El programa muestra una boleta simple de venta con:

- nombre de la tienda
- nombre del cliente
- nombre del producto
- cantidad
- precio unitario
- subtotal
- IGV
- total a pagar

## Tecnologías usadas

- Java
- NetBeans IDE 8.1
- JDK 7u80
